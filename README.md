### ✨ Modern AI-Powered Developer Portfolio

A highly interactive, futuristic, and performance-optimized portfolio website built with **Next.js 16, React 19, TypeScript, Tailwind CSS, Framer Motion, and AI integrations**.

Designed to showcase projects, skills, achievements, analytics, and professional experience through an immersive user experience.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/yourusername/portfolio.git
```

## Navigate

```bash
cd portfolio
```

## Install Dependencies

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```

Open:

```txt
http://localhost:3000
```
